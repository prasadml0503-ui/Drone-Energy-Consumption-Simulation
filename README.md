# Drone Energy System Simulation

## Overview
A Python-based simulation model to estimate hover power requirements and analyze battery consumption using aerodynamic and physical parameters.

## Features
- Hover power estimation
- Battery life prediction
- Parameter-based simulation
- Lightweight analytical model

## Working Principle
The model calculates thrust, drag, and power consumption using standard aerodynamic equations. It evaluates how parameters like mass, propeller area, and efficiency affect energy usage.
